Universal Selector

  • It is applicable to all elements and tags including html and head tag
  • It is denoted by using "*" symbol
  • It is also called as global style
  • For Exapmle taking Background as Aqua

Tag Selector

  • It is applicable to only selected tags those you want to apply styles
  • Same style is applied to all same tag elements usig tag selector
  • All Headings are colored with red are by using tag selector

Id Selector

  • It is used to apply specific style to specific element then we use id selector
  • In this, we use a specific and unique id to style the tag and elements
  • We can use a same id to style different element but it is bad practise
  • If we want to make Changes on 1 but due to same id Changes on all tags takes place due to same id so it should be avoided
  • One of the line is different from this is due to id selector
  • It is denoted by using "#" symbol

Class Selector

  • Different elements of different tags are styled through class tag
  • It is represented by using "." symbol
  • Different tags are styled with same style using CSS classes
  • Header and one line of this topic is changed using Class selector

Grouping selector

  • If we want to give same style to more than one tag then we use Grouping selector
  • Grouping selector gives same styles to multiple tags
  • In this same tag with different name should be used
  • It is represented by using "," symbol

Child selector

  • This is used to style all the childs of parent and grandparent tags
  • By using this we can add more specificity to the tag that we want to style
  • It is represented by using " " or space symbol
  • All of the line from this are colored using this child selector

Direct Child

  • It selects only direct child from parent or grandparent rather than other indirect childs
  • It is used to access only direct child i.e. tag inside tag only
  • It is represented by using ">" symbol
  • All of the line can be changed is due to this direct child selector
